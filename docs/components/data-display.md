# Data Display Components

Componentes para mostrar información y contenido al usuario.

---

## Accordion

- **Archivo:** `accordion.tsx`
- **Descripción:** Contenido colapsable con ítems expandibles (título + contenido).
- **Exports:** `Accordion`, `AccordionItem`, `AccordionTrigger`, `AccordionContent`.
- **Usado por:** (ningún otro componente UI)

## Avatar

- **Archivo:** `avatar.tsx`
- **Descripción:** Imagen de avatar con fallback de texto o icono si la imagen falla.
- **Exports:** `Avatar`, `AvatarImage`, `AvatarFallback`.
- **Usado por:** (ningún otro componente UI)

## Badge

- **Archivo:** `badge.tsx`
- **Descripción:** Etiqueta visual pequeña para estados, categorías o contadores.
- **Exports:** `Badge`, `badgeVariants`.
- **Usado por:** (ningún otro componente UI)

## Card

- **Archivo:** `card.tsx`
- **Descripción:** Contenedor con secciones de header, content y footer.
- **Exports:** `Card`, `CardHeader`, `CardFooter`, `CardTitle`, `CardDescription`, `CardContent`.
- **Usado por:** (ningún otro componente UI)

## Carousel

- **Archivo:** `carousel.tsx`
- **Descripción:** Slider de contenido con navegación, usa Embla Carousel.
- **Exports:** `Carousel`, `CarouselContent`, `CarouselItem`, `CarouselPrevious`, `CarouselNext`.
- **Usado por:** (ningún otro componente UI)

## Chart

- **Archivo:** `chart.tsx`
- **Descripción:** Gráficos basados en Recharts con contenedor y tooltip integrados.
- **Exports:** `ChartContainer`, `ChartTooltip`, `ChartTooltipContent`, `ChartLegend`, `ChartLegendContent`.
- **Usado por:** (ningún otro componente UI)

## Empty

- **Archivo:** `empty.tsx`
- **Descripción:** Estado vacío con icono y texto para listas o tablas sin datos.
- **Usado por:** (ningún otro componente UI)

## Item

- **Archivo:** `item.tsx`
- **Descripción:** Elemento de lista con título, descripción e icono opcional.
- **Usado por:** (ningún otro componente UI)

## Table

- **Archivo:** `table.tsx`
- **Descripción:** Tabla HTML estilizada con header, body, rows y celdas.
- **Exports:** `Table`, `TableHeader`, `TableBody`, `TableFooter`, `TableHead`, `TableRow`, `TableCell`, `TableCaption`.
- **Usado por:** (ningún otro componente UI)
